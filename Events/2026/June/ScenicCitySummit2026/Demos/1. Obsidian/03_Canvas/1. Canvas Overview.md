# Canvas — Your Infinite Whiteboard

Obsidian Canvas is a **free core plugin** that turns any idea into a visual workspace. Unlike a note, a Canvas has no top or bottom — just infinite space to arrange, connect, and think.

---

## What Can Go on a Canvas?

| Element | How to Add | Use It For |
|---|---|---|
| **Text card** | Double-click empty space | Quick ideas, labels, annotations |
| **Note card** | Drag a `.md` file onto the canvas | Embed living notes in your map |
| **Media** | Drag image/PDF onto canvas | Screenshots, diagrams, reference docs |
| **Group** | Select cards → right-click → Create group | Cluster related ideas visually |
| **Arrow/Edge** | Hover card edge → drag the dot | Show relationships and flow |

---

## Demo 1 — Fallout Universe Map

**Open:** `Fallout Universe Map.canvas`

This canvas maps the entire Fallout lore as a connected visual:

- **Note cards** — each card is a live `.md` file. Click the arrow icon to open the full note.
- **Groups** — factions, games, and TV series are visually clustered.
- **Edges** — arrows show relationships (e.g., Vault-Tec → individual Vaults).

> **Show the audience:** Double-click a note card to edit it inline — the change saves back to the original file instantly.

## Demo 2 — DP-700 Study Map ⭐ Most Relevant for Data Pros

**Open:** `DP-700 Study Map.canvas`

This is a real-world data professional use case — a **certification study map** for the Microsoft Fabric Analytics Engineer exam.

**Walk through these highlights:**

1. **Hub-and-spoke layout** — the DP-700 card at center connects to all four exam domains with weighted edges (10–15%, 40–45%, etc.)
2. **Color-coded groups** — each domain has its own color for instant visual orientation
3. **Dense text cards** — DAX essentials, Medallion architecture, KQL syntax all captured in one view
4. **Resources group at the bottom** — study links, practice exams, and a 12-week plan in one place

> **Key talking point:** You can build this kind of map for *any* complex topic — a data pipeline architecture, a client's system design, a project kickoff plan.

## Why Data Professionals Love Canvas

- **Architecture diagrams** — sketch out your data pipeline or star schema visually, with linked notes for each component
- **Project planning** — map out workstreams and connect them to actual task notes
- **Stakeholder maps** — visualize who owns what across a data platform
- **Study & certification prep** — exactly what the DP-700 map demonstrates
- **Meeting prep** — arrange context cards before a big presentation or discovery call

---

## Tips to Show Live

- **Zoom out** with `Ctrl+Scroll` to show the full map at a glance
- **Fit to screen** with `Ctrl+Shift+F` — great for presenting
- **Drag a note from the file explorer** directly onto the canvas to embed it
- **Right-click any card** to see options: convert to file, create group, change color
