# Daily Notes + Calendar — Your Work Journal

Daily Notes is a **free core plugin** that creates one dated note per day with a single keystroke. Paired with the **Calendar** community plugin, it becomes a visual work journal you can navigate like a calendar.

---

## Why Data Professionals Use This

Most data work involves context-switching — pipelines, meetings, stakeholder requests, study goals. Daily notes give you a single place to capture everything without building a folder structure. Over time your vault becomes a searchable log of your entire work history.

---

## Demo 1 — Open Today's Note

**`Ctrl+Shift+D`** — creates (or opens) today's daily note instantly.

The note is auto-named by date (`YYYY-MM-DD`) and placed in your `Daily Notes/` folder. No manual filing needed.

> **Show the audience:** Press `Ctrl+Shift+D` live. A blank note appears — ready to type.

**Open:** `02_Templates/Daily Note Template.md`

## Demo 4 — Calendar Plugin

The **Calendar** plugin adds a calendar widget to the right sidebar. Each day with a note gets a dot — click any date to open or create that day's note.

> **Show the audience:** Click on a past date in the calendar. It either opens an existing note or creates a new one from the template. No hunting through folders.

## Connecting It All with Dataview

Because daily notes have consistent frontmatter (`type: daily-note`, `date:`), you can query them like a database. Example — add this to any note to see your recent log:

```dataview
TABLE date, file.name AS "Note"
FROM ""
WHERE type = "daily-note"
SORT date DESC
LIMIT 7
```

## Setup (For Your Own Vault)

1. **Enable Daily Notes** — Settings → Core Plugins → Daily Notes ✅
2. **Set template** — Daily Notes settings → Template file → `02_Templates/Daily Note Template`
3. **Set folder** — Daily Notes settings → New file location → `Daily Notes`
4. **Enable Calendar plugin** — Settings → Community Plugins → Calendar ✅
