# toolkit-obsidian-demo

Obsidian vault used as the demo for the "Data Professional's Power Toolkit" session. Built around Fallout universe content to demonstrate Obsidian's core features in a way that's engaging for a technical audience.

Open the root of this repo as your Obsidian vault.

## Vault structure

- Root — Fallout content notes (characters, factions, games, items) — the interconnected data that powers the Graph View demo
- `00._Navigation/` — demo steps: Files, Search, Bookmark
- `01_Graphview/` — demo steps: Graph View and settings
- `02_Templates/` — Character, Faction, Game, and Daily Note templates
- `03_Canvas/` — Canvas demo
- `04_Dataview Dashboard/` — Dataview plugin demo
- `05_Plugins/` — Git and Excalidraw plugin demos
- `06_Daily Notes/` — Daily Notes demo
- `Index.md` — starting point for the demo

## Demo order

Follow the numbered folders in sequence. Root-level Fallout notes are the content; numbered folders are the guided walkthrough.

## Notes

- Daily Notes plugin should be configured to write to `Daily Notes/`
- Obsidian Git plugin is included for the Git demo step
